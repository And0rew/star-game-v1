 Как запустить сервер
    1. Открыть терминал и написать node -v и проверить установлена ли нода. Если установлена — то пропустить следующий ход.
    2. Если не установлена — установить ноду из https://nodejs.org/ru/ желательно чётной версии
    3. Зайти в терминале в папку start-game-server
    4. Выполнить команду npm i
    5. Выполнить команду node star-game-server.js для запуска сервера

Куда класть файлы
    HTML и JS файлы самой игры класть в папку star-game-www.
    Например, мы сделали файл game.html
    Если сервер успешно запустился — то локально можно увидеть файл
        http://localhost:3000/game.html

    С другого компьютера в сети его можно увидеть через IP адрес
        http://ip_address:3000/game.html
